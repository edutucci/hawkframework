## :computer: Installation
**yarn**

```bash
yarn add hawkvuemd
```

**npm**
```bash
npm install hawkvuemd
```
# Usage
In your project add the code below in your main.js

```js
import 'hawkvuemd'
import 'hawkvuemd/dist/hawk_default_stylus.styl'
import 'hawkvuemd/dist/hawkvuemd.css'
```
